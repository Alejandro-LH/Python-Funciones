# Fundamentos de Funciones en Python

Este repositorio reúne una serie de notebooks de Jupyter con ejemplos y ejercicios sobre funciones en Python. El material está organizado como una guía práctica para estudiar conceptos básicos e intermedios, con explicaciones paso a paso y fragmentos de código ejecutables.

## Contenido del repositorio

- [code01.ipynb](/Users/alejandrolh/Downloads/code01.ipynb): introducción a las funciones, definición, ventajas, ámbito de variables, declaración, llamada de funciones y uso de parámetros y argumentos.
- [code02.ipynb](/Users/alejandrolh/Downloads/code02.ipynb): valores de retorno, uso de `return`, funciones que regresan `None`, y manejo de parámetros predeterminados mutables e inmutables.
- [code03.ipynb](/Users/alejandrolh/Downloads/code03.ipynb): funciones como objetos, paso de funciones como argumentos, funciones `lambda`, recursión, cálculo de factorial y evaluación básica de rendimiento.
- [code04.ipynb](/Users/alejandrolh/Downloads/code04.ipynb): documentación con docstrings, comentarios en código, organización en módulos, importación de funciones, pruebas unitarias y uso de `assert`.

## Temas principales

- Definición y uso de funciones
- Parámetros, argumentos y argumentos nombrados
- Alcance de variables y namespaces
- Uso de `return`
- Parámetros predeterminados y buenas prácticas
- Funciones como objetos
- Funciones anónimas con `lambda`
- Recursión
- Documentación con docstrings
- Modularización del código
- Pruebas y depuración

## Objetivo

El objetivo de este repositorio es servir como material de apoyo para aprender y practicar programación en Python, especialmente en el tema de funciones y organización básica del código.

## Cómo usar este repositorio

1. Abrir los notebooks en Jupyter Notebook o JupyterLab.
2. Ejecutar las celdas en orden para seguir la explicación y observar los resultados.
3. Modificar los ejemplos para experimentar con otros valores y entender mejor cada concepto.

## Requisitos

- Python 3
- Jupyter Notebook o JupyterLab

Si no tienes Jupyter instalado, puedes usar:

```bash
pip install notebook
```

o bien:

```bash
pip install jupyterlab
```

## Nota

Algunos notebooks incluyen ejemplos diseñados para mostrar errores comunes o comportamientos importantes de Python. En esos casos, el propósito es didáctico.
